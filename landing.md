# TypeBetter - AI Text Rephrasing App for macOS

## What is TypeBetter?

TypeBetter is a lightweight macOS menu bar app that lets you instantly rephrase, fix, or transform any selected text using AI. Simply select text anywhere on your Mac, press a hotkey, and tell the AI what to do with it. The rephrased text automatically replaces your selection.

---

## Core Features

### Instant Text Transformation
- Select any text in any app on your Mac
- Press a global hotkey to open the TypeBetter popup
- Type what you want (e.g., "make it formal", "fix grammar", "simplify")
- AI processes your request and replaces the selected text automatically

### Multiple AI Providers
- **Claude (Anthropic)** - Advanced reasoning and natural language
- **OpenAI (GPT-4o)** - Fast and reliable text processing
- **Ollama** - Run AI locally on your Mac for complete privacy

### Quick Action Buttons
- Pre-configured one-click actions for common tasks
- Default actions include: Rephrase, Fix Grammar, Shorter, Longer, Formal, Casual
- **Keyboard shortcuts (1-9)** - Press number keys to instantly trigger actions
- Fully customizable - create your own buttons with custom prompts
- **Color coding** - Assign colors to buttons for visual organization (Purple, Blue, Green, Orange, Pink, Red, Yellow, Teal)

### Built-in Translation
- Translate selected text to any language with one click
- Supports: Spanish, French, German, Italian, Portuguese, Chinese, Japanese, Korean, Russian, Arabic, Hindi, and more

## Privacy & Security

### Secure API Key Storage
- API keys stored in macOS Keychain (not plain text)
- Same security used by system passwords
- Keys never leave your device

### Local AI Option
- Use Ollama to run AI models entirely on your Mac
- Zero data sent to external servers
- Complete privacy for sensitive documents

### iCloud Sync
- Quick action buttons sync across all your Macs
- Settings preserved across app reinstalls
- Seamless experience on multiple devices

## System Integration

### Global Hotkey
- Customizable keyboard shortcut (default: F16)
- Works in any application

### Launch at Login
- Option to start TypeBetter automatically when you log in
- Always ready when you need it
- Uses native macOS login items
